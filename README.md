## 运行

使用下面的命令在本地启动这个项目.

```bash
git clone https://github.com/qwe1356879/knowledge-base.git

cd knowledge-base

npm install

npm run dev
```

Visit [https://localhost:5173](https://localhost:5173) to see your site

## 打包
You may run this command to build the docs

```bash
npm run build
```
